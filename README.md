# 2D Platformer Game

Welcome to the 2D Platformer project! Follow the steps below to download, set up, and run the game on your computer using Unity.

---

## 🚀 Getting Started

### 1. Download and Install Unity
Before opening the project, you need Unity Hub and the correct editor version:
* Download and install [Unity Hub](https://unity.com/download).
* Install **Unity 6.4** (or the specific version matching this project) via the Unity Hub installs tab.

### 2. Clone the Repository
Clone this project to your local machine using Git Bash or your preferred Git desktop application:
```bash
git clone https://github.com/Fallen-coder/Platformer_Unity.git
